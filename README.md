# Hakumonogatari-tools
怪談白物語 GMサポートツール 操作手順マニュアル
全体のドキュメントから操作手順のみを抽出し、他の要件や仕様の項番から切り離して、**操作手順だけで独立した通し番号（1〜9）**のマークダウン形式にまとめ直しました。
Ⅰ. シナリオ準備・管理編
 * シナリオの登録
   * 操作内容: 画面最下部の「シナリオ新規登録」フォームで、タイトル・本文（段落間は空行必須）・キーワード（カンマ区切り）を入力し保存します。
 * シナリオの編集
   * 手順1: 「進行するシナリオ」ドロップダウンまたは検索ボックスから対象を選択します。
   * 手順2: **「選択中のシナリオを編集」**ボタンを押すと、入力フォームに現在の内容が読み込まれます。
   * 手順3: 内容を修正し「シナリオを保存する」を押すことで上書きされます。
 * シナリオの削除
   * 操作内容: 削除したいシナリオを選択し、**「選択中のシナリオを削除」**を押します（確認ダイアログが出ます）。
 * データのバックアップ
   * 操作内容: 準備完了後、**「データをファイルに保存」**を押してJSONファイルを書き出し、保管してください。
Ⅱ. セッション本番編
 * ポップアウトによる画面共有・配信の最適化
   * 手順1: 物語の表示エリア内にある 「ポップアウト表示」 ボタンを押します。
   * 手順2: 余計なヘッダーやGM操作パネルをすべて排除した、プレイヤー共有専用の別ウィンドウが新しく立ち上がります。
   * 手順3: DiscordやZoomなどの通話ツール、またはOBS等の配信ソフトで画面を共有する際は、メインのGM画面ではなく、このポップアウトしたウィンドウ（ミラー画面）だけをターゲットに指定してキャプチャ・共有してください。これにより、GM手元のネタバレ領域がプレイヤーに誤って映り込む事故を100%防止できます。
 * 物語の進行
   * 進行: GMパネルでシナリオを選択し、**「次の段落へ ＞」**を押して文章を公開・読み上げます。
   * 同期: ボタンを押すと同時に、ポップアウトウインドウ側へもノータイムで文章が同期され、段落が枠を超えた場合は自動で下部へ追従スクロールします。
   * 巻き戻し: 万が一進めすぎた場合は、**「＜ 前の段落へ」**を押すことで公開状態を1つ前に戻せます（ポップアウト側も即座に巻き戻ります）。
 * キーワードの改変
   * 登録: プレイヤーがキーワード書き換えに成功した場合、「対象キーワード」と「改変後の言葉」を入力して**「改変登録」**を押します。
   * 反映: 即座に親画面・ポップアウト画面の双方が連動し、該当する単語が綺麗な黄色いハイライト付きで置き換わり、進捗メーターが更新されます。
 * 改変の修正・取り消し機能
   * 取消: 「改変後の言葉」の入力ミスや、GMの裁定見直しが発生した場合、「現在の改変履歴」リストから該当する項目の「解除」ボタンを押します。
   * 修正: 即座に改変が取り消され、すべての画面の単語が元の状態にリアルタイム復元されます。復元後、正しい言葉で再度「改変登録」を行ってください。
 * セッションの中断と再開
   * 中断: 途中で中断する場合は、「データをファイルに保存」でエクスポートします。
   * 再開: 再開時は「データファイルを読み込み」でファイル（新旧どちらの形式でも可）を指定すれば、途中の進捗や改変履歴、ポップアウトウィンドウへのミラーリング状態を含めて完全復元されます。
