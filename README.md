# Hakumonogatari-tools
白物語用ツール

操作手順（シナリオ準備編）
ゲームセッションを行う前の、GMによる事前準備の手順です。
1. ツールの起動: 公開したGitHub PagesのURL（またはローカルの ⁠index.html⁠ ）をブラウザで開きます。
2. シナリオの新規登録: 画面最下部の「シナリオ新規登録」フォームを使用します。
 タイトル: シナリオの題名を入力します。
 本文: 怪談の本文を入力します。【重要】後述の逐次表示のため、段落と段落の間には必ず1行の「空行（エンターキー2回）」を挟んでください。
 ターゲットキーワード: 白物語のクリア条件（または防御対象）となる単語を、半角カンマ区切りで入力します（例: ⁠人形,夜中,鏡⁠ ）。
3. 登録の実行: 「シナリオを登録する」ボタンを押すと、上の「進行するシナリオ」セレクトボックスに自動で追加されます。
4. データのバックアップ（推奨）: シナリオを入力し終えたら、操作パネル内の「データをファイルに保存」ボタンを押し、バックアップJSONファイルをPCに保存しておきます。
 これにより、ブラウザのキャッシュがクリアされても、次回「データファイルを読み込み」から瞬時にシナリオを復元できます。

操作手順（セッション本番編）
実際のゲームセッション中のゲームマスター（GM）の操作手順です。
1. 画面共有のセットアップ:
 DiscordやZoom等の通話ツールで画面共有を開始します。
 共有する際は「画面全体」ではなく「ウィンドウ指定」または「ブラウザのタブ指定」を選択し、さらにブラウザを適度にスクロールして**「物語の表示（画面共有用）」の黒枠エリアだけがプレイヤーに見えるように調整**します。
 ※「GM専用：操作パネル」やキーワードリストが映るとネタバレになるため注意してください。
2. シナリオの選択: 「進行するシナリオ」から、本日プレイするシナリオを選択します。
3. ゲーム開始: 「次の段落へ ＞」ボタンを押すと、最初の段落が画面共有エリアに表示されます。GMは表示された文章を読み上げます。
4. プレイヤーによる改変（サイコロ成功時）:
 プレイヤーがダイスに成功し、キーワードを別の言葉に書き換えたら、すかさずGM操作パネルの「対象キーワード」と「改変後の言葉」に入力し、「改変登録」ボタンを押します。
 （例：対象キーワードに「人形」、改変後の言葉に「タワシ」と入れて登録）
 効果: 画面共有エリア内の該当する単語がすべて自動的に黄色くハイライトされ、「改変後の言葉」に置き換わります。同時に、GMパネル内の「進捗」がカウントアップされ、どのキーワードが攻略済み（打ち消し線表示）かが一目でわかります。
5. 物語の進行: プレイヤーとのやり取りが一区切りついたら、再び「次の段落へ ＞」を押し、次の文章を公開・読み上げていきます。
6. セッションの中断・終了:
 万が一途中で中断する場合や、セッションの全記録を残したい場合は、「データをファイルに保存」をクリックして状況を書き出してください。
 次回、そのファイルを「データファイルを読み込み」から選択すれば、どの段落まで進んでいたか、どのキーワードが何に改変されていたかを含めて、完全にセッションを再開できます。
